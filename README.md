# FitPulse — приложение для похудения

Веб-приложение для отслеживания питания, тренировок и прогресса веса.

**Живой сайт:** https://4rakhman28-stack.github.io/app10/

## Запуск локально

```bash
npm install
npm run dev
```

Открой **http://localhost:5173/**

> Не открывай `index.html` двойным кликом — будет пустая страница.

## Возможности

- Дашборд с калориями, водой и прогрессом
- Дневник питания
- Тренировки
- График веса
- Профиль пользователя

## Стек

- React 19 + TypeScript
- Vite + Tailwind CSS
- Recharts + Lucide Icons
